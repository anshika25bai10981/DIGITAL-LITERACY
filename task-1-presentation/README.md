## Digital Literacy Infographic


This project presents a visual infographic designed to spread awareness about digital literacy among students. It highlights key concepts in a simple, engaging, and easy-to-understand format.

## Topics Covered
 -- What is Digital Literacy
 -- Useful Digital Tools for Students
 -- Safe Internet Practices

 
## Objective

To help students understand how to effectively use digital tools, stay safe online, and develop responsible digital habits.
